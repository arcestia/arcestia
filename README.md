## 💻 What I'm Working On

I love building robust systems and exploring new technologies. Here are a few things currently taking up my compute time:

* **Specter:** A domain status monitoring tool built with Golang and PostgreSQL.
* **Axon:** An AI-powered chess engine I'm developing to combine my interest in machine learning and the game itself.
* **Infrastructure & Homelab:** Managing a high-capacity Proxmox cluster (featuring a 60-bay Supermicro storage server and a heavy-duty Dell compute node).
* **Paper Pulse:** Exploring my creative side with my fiancée through our creative duo.
